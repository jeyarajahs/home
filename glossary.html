Computer Studies Defintions

Application software

A program or a group of programs designed to perform a function for the benefit of the user of the program.

ASCII

ASCII is an abbreviation for the American Standard Code of Information Interchange. ASCII character encoding is a method of providing a standard way to represent characters using numeric codes.

Binary Code

A system of coding in which the values 0 or 1 are used to represent text, processor information, digits, spacing, etc.

Bit

The smallest unit of data in a computer which is given an individual, binary value of 1 or 0.

Booting

The sequence of commands which occurs when an operating unit is turned on, such as a computer.

Byte

In most computer systems, a byte is a unit of data that is eight binary digits long. A byte is the unit most computers use to represent a character such as a letter, number or spacing.

Character

A character is any letter, number, space, punctuation mark, or symbol that can be typed on a computer.

Computer

An electronic device used for the purpose of storing and processing data, typically in binary form, according to instructions given to it.

CPU

An abbreviation for central processing unit, commonly called central processor. This is a component is necessary for a operating unit to function.

Directory

A catalog structure containing links to other computer files or possibly other directories.

File

A collection of information and data stored under a specific name, commonly referred to as a file name.

Firewire

A technology that allows exchange between a computer and a peripheral or between two computers at high-speed communication and data exchange.

Gigabyte

A unit of information equal to one billion (10 to the power of 9)

Hardware

Computer hardware is the collection of physical parts which give input, allowing a computer system to function.

Information

Processed, stored, or transmitted data

Input Devices

An input device is a piece of computer hardware equipment used to provide data and control signals to an information processing system such as a computer or information appliance.

Kilobyte

A unit of memory used to represent 1000 bytes.

Megabyte

A unit of memory used to represent 1000000 bytes (10 to the power of 6, bytes)

Monitor

A display screen used to provide visual output from a computer, cable box, video camera, VCR or other video generating device.

Operating System

An operating system (OS) is system software which is responsible for managing computer hardware and software resources and offers essential services for computer programs.

Output devices

A device which takes information from a computer or another device to another device, such as a printer, projector, speakers, headphones.

Peripheral

Another word for hardware is something that can be used to create input.

Primary storage

A part located in the computer where data can be stored for quick access storage. The abbreviation “Random Access Memory” (RAM) are often referred to as synonyms to primary or main storage.

Processing

The action of a computer program being executed.

Programmable

An adjective used to describe the ability to provide a code of instruction which will be able to perform the task automatically.

Program

A computer program is a collection of instructions that performs a specific task(s) when user executes it by a computer.

RAM

An abbreviation for Random Access Memory where any byte of memory can be accessed without touching the preceding bytes.

ROM

Read only memory, it is computer memory of data that has been pre recorded.

Root directory

Root commonly refers to the highest directory in a file system. For instance, on a Windows-based PC, “C:\” demonstrates that the root directory is located in the C drive.

Secondary storage

Secondary storage is known as external memory which the computer usually uses its input/output channels to access secondary storage and transfers the desired data using intermediate area in primary storage.

Software

Software is a general term used to describe kinds of programs used to operate computers and related devices.

String

A string is a definite sequence of characters, such as spacing, numbers, letters

System Unit

A system unit is the part of a computer that contains the primary devices which are responsible for operations that produce results.

USB

USB also referred to as a Universal Serial Bus is used to enable communication between devices.
